Identifying Targets

The following 3 methods are specifically for vulnhub vm boxes which are hosted on your Host machine.

Method 1

Use "sudo arp-scan -I vboxnet0 192.168.56.0/24" to discover the device, If you are not able to identify the vm by running "sudo arp-scan -I vboxnet0 192.168.56.0/24" and if multiple devices are connected, run "sudo arp-scan -I vboxnet0 192.168.56.0/24" and then turn on the machine and rerun arp-scan to get the IP which was not shown earlier.

Method 2

Import vm into vmware/vbox and set network adapter to be in Host-Only mode. use "ifconfig" in base to check vmnet ip and run nmap ping scan on the range "nmap -n -sn -vvv 192.168.136.0-255".

Method 3

If we know the services running on the vm, then we can use "nmap -sS -Pn 192.168.1.0/24" to scan open ports and ip address which can be compared with known services.
