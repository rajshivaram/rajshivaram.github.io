Access Control Fundamentals

Access Control Models

Controlling access means regulating who is authorized to take a particular action, whether viewing/modifying files or accessing certain physical locations.

Popular Methods of Implementing Access Control Policies

  • Mandatory Access Control (MAC)
  • Discretionary Access Control (DAC)
  • Attribute-Based Access Control (ABAC)
  • Role-Based Access Control (RBAC)
  • Rule-Based Access Control

Mandatory Access Control

A means of restricting access to system resources based on the sensitivity (as represented by a label) of the information contained in the system resources and the clearance of users to access that information.
Users are given an access level and are only allowed to view object at (and sometimes below) their access level.


Discretionary Access Control

A means of restricting access to objects where the owner of the object has full control and is able to change the permissions of other users as they pertain to the object and its data.
The controls are discretionary in the sense that accessibility is changed at the owner’s discretion.

Attribute-Based Access Control

  • An acccess-control approach in which access is given based on attributes associated with requesters and the objects to be accessed.
  • Each object and subject has a set of associated attributes, such as location, time of creation or access rights and access to an object is granted or denied depending upon whether the attributes of the requester match those required for access.
  • Provides a greater selection of control options than other access control methods.

Role-Based Access Control

  • A model for controlling access to resources where actions are permitted on resources based on requester roles rather than individual identities.
  • Similar to the DAC model, but with more flexibility. Only administrators can modify access permissions, and users are assigned to various groups known as “roles” which each provide the right to access certain files.

Rule-Based Access Control

A security policy which is based on global rules determined by a system-wide policy and imposed for all subjects is a type of rule-based access control.

  • Individual users are not able to change object permissions, and must follow the policy rules.
  • MAC, RBAC (Role-Based Access Control) and ABAC can all be considered methods of Rule-Based Access Control.

Biometric Factors

Biometric factors such as fingerprints, iris/retina and facial scans and voice recognition can be used to authenticate users and control access.

While often quite reliable, these devices do sometimes fail, resulting in either an authorized party being denied access – or worse, an unauthorized attacker being allowed to access a protected resource.

Tokens

Hardware and Software Tokens

Hardware tokens are physical devices, such as a key or a smart card, which provide a user with access to a resource.

  • Software tokens are generated by an application (such as Google Authenticator) and generally consist of a string of letters/numbers which must be used as input to authenticate.
  • Tokens pose an increased risk of loss/theft, and are generally used in conjunction with other authentication factors.

Certificate Based Authentication

  • Authentication can also be granted through the use of certificates. These certificates can be built into smart devices, removing the requirement of a user having to remember a password or authenticate using a biometric.
  • Known as the Port-based Network Access Control standard, IEEE 802.1x describes the many ways to perform port-based certificate authentication before allowing access to a network.

Credential Management and Enforcement

  • Having a set of policies to make sure methods of authentication remain secure is critical.
  • Users need to be trained not to divulge their passwords and to keep any physical access tokens secure at all times.
  • Passwords should be required to be complex, and if they must be transmitted or stored, hashing or strong encryption should be required.
  • This can seem like a daunting task in a large environement, but there are tools to assist.
